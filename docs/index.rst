================
Tuatha de Danann
================

Dana is the server which support connexion of players and which maintain a representation of the virtual world.
Most of game events, actions and verification are managed by Dana. The client (Etain) is basically devoted to graphics displaying. All game events are calculated by Dana and then transmitted to Etain which has only to display it.
